# 📚 Образовательная платформа

## 📌 Описание проекта

Веб-приложение для создания и прохождения онлайн-курсов.
Пользователи могут просматривать курсы, изучать уроки и проходить тесты.
Администратор управляет контентом через защищённую панель.

---

# 🧭 Этапы разработки

## 1. Анализ требований

На первом этапе были определены основные функции системы:

* просмотр списка курсов
* просмотр уроков внутри курса
* прохождение тестов
* административная панель
* загрузка изображений
* поиск и пагинация

Также было принято решение использовать:

* Go (backend)
* PostgreSQL (база данных)
* HTML/CSS/JS (frontend)
* Docker (развертывание)

---

## 2. Проектирование архитектуры

### Общая архитектура:

* Backend: REST API на Go
* Frontend: статические страницы + JS
* База данных: PostgreSQL
* Контейнеризация: Docker + docker-compose

### Основные сущности:

* Course (курс)
* Lesson (урок)
* Test (тест)
* Question (вопрос)
* Answer (ответ)

---

## 3. Проектирование базы данных

Были созданы следующие таблицы:

* `courses`
* `lessons`
* `tests`
* `questions`
* `answers`

Связи:

* курс → уроки (1:N)
* урок → тест (1:1)
* тест → вопросы (1:N)
* вопрос → ответы (1:N)

---

## 4. Реализация backend

Реализованы REST endpoints:

### Курсы

* `GET /courses`
* `POST /courses`
* `GET /courses/{id}`

### Уроки

* `GET /lessons`
* `POST /lessons`
* `GET /lessons/{id}`

### Тесты

* `POST /tests/{lesson_id}`
* `GET /tests/{lesson_id}`

### Отправка теста

* `POST /submit/{test_id}`

### Загрузка файлов

* `POST /upload`

---

## 5. Реализация frontend

Созданы страницы:

* `/` — список курсов
* `/course` — список уроков
* `/lesson` — урок + тест
* `/admin` — админ-панель

Использован:

* Vanilla JavaScript
* Fetch API
* Динамическая генерация HTML

---

## 6. Реализация административной панели

Функции:

* создание курсов
* создание уроков
* создание тестов
* загрузка изображений

Добавлена защита:

* HTTP Basic Auth для `/admin`

---

## 7. Дополнительный функционал

### 🔍 Поиск

Реализован поиск по названию и описанию курсов.

### 📄 Пагинация

* ограничение количества курсов на странице
* переключение страниц

### 🖼 Изображения

* загрузка через `/upload`
* хранение URL в БД
* отображение в уроках

### 🌙 Темная тема

* переключатель темы
* сохранение в `localStorage`

Проект запускается через Docker:

* backend (Go)

Используется `docker-compose`.
